Как связать с Telegram-ботом

GitHub Pages не должен хранить токен Telegram. Значит связка делается через отдельный backend.

Схема: 1) сайт на GitHub Pages показывает интерфейс 2) backend хранит последний сигнал и токен Telegram 3) Telegram-бот принимает команду или сам рассылает сигнал 4) сайт делает fetch к backend и показывает этот сигнал на стриме
Этот интерфейс — демонстрационный и контентный. Он нужен для стримов и вовлечения аудитории. Не является финансовой рекомендацией.
